# EZTerminal 1.0.1

Node 설치나 외부 요인으로 셸 인터프리터 프로세스가 종료되어도 터미널 입력이 영구 정지하지 않도록 복구 기능을 추가한 안정화 릴리스입니다.

## 주요 수정

- 셸 인터프리터가 예기치 않게 종료되면 제한된 지연으로 자동 재시작합니다.
- 기존 세션 ID와 마지막 작업 경로를 새 인터프리터에 복원합니다.
- 중단된 명령은 오류 상태로 명확히 정리하고 키보드 입력을 다시 활성화합니다.
- 이전 인터프리터의 늦은 비동기 응답이 새 프로세스로 전달되지 않도록 세대 검사를 추가했습니다.
- 복구 상태와 로컬 `main.log` 경로를 앱 배너에 유지합니다.

## 다운로드

- Windows 10 22H2 / Windows 11 x64: `EZTerminal-Setup.exe`
- Android 10(API 29) 이상: `EZTerminal-Android-1.0.1-vc22.apk`
- 무결성 검증: `SHA256SUMS.txt`

Windows 빌드는 코드 서명되지 않아 SmartScreen 경고가 표시될 수 있습니다. Android APK는 1.0.0과 같은 장기 릴리스 키로 서명되어 기존 1.0.0 설치 위에 업데이트할 수 있습니다.
